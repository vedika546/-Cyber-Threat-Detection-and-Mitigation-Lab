# 🛡️ Cyber Threat Detection and Mitigation Lab

> **Documentation of a cybersecurity laboratory focused on enterprise threat detection, network traffic analysis, incident response, and security operations using enterprise security concepts.**

---

## 📖 Overview

This repository documents a cybersecurity laboratory that focuses on understanding how Security Operations Center (SOC) teams detect, investigate, and respond to security incidents in enterprise environments.

The objective of this lab was not to develop a software application, but to gain practical exposure to enterprise security tools, network traffic analysis, threat investigation, and incident response methodologies.

The lab involved simulated attack scenarios, firewall monitoring, packet analysis, cloud security concepts, and security workflow documentation.

---

# 🎯 Objectives

The primary objectives of this laboratory were to:

- Understand enterprise threat detection workflows
- Analyze suspicious network traffic
- Investigate security events
- Study firewall-based threat detection
- Perform packet analysis using Wireshark
- Understand cloud security monitoring concepts
- Learn incident response methodologies
- Map attack techniques using the MITRE ATT&CK Framework

# 🚨 Incident Response Workflow

The investigation process followed a structured Security Operations Center (SOC) workflow:

1. Alert Detection
2. Alert Validation
3. Evidence Collection
4. Network Traffic Analysis
5. IOC Identification
6. Threat Containment
7. Incident Documentation
8. Post-Incident Review
